# 柳工报销文档生成器 / LiuGong Expense Generator

填写表单 → 一键生成三份报销文档（Смета / приказ / АктОтчет）

## 使用方法

1. 打开网页，填写报销信息
2. 点击"生成报销文件"
3. 下载 ZIP 包或单独下载三份文件

## 部署到 Streamlit Cloud

1. 将此仓库上传到 GitHub
2. 在 [share.streamlit.io](https://share.streamlit.io) 登录
3. 选择此仓库，点击 Deploy

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```
